# STYLE GUIDE

## Naming Conventions
- **Files/Folders**: `kebab-case` (e.g., `src/entities/npc-manager.ts`)
- **Variables/Functions**: `camelCase`
- **Classes/Interfaces/Types**: `PascalCase`
- **CSS Classes**: `kebab-case`

## Code Standards
- **Line Length**: Max 100 characters.
- **Function Size**: Max 50 lines.
- **File Size**: Max 400 lines (Refactor `src/main.ts` soon).
- **Trace Tags**: `// [TRACE: filename.md]`
- **Comments**: Focus on "WHY", not "WHAT".

## Documentation
- Use `DOCS/` for all project-level documentation.
- Update `SCRATCHPAD.md` and `CHANGELOG.md` regularly.
- Maintain `SBOM.md` for dependencies.
