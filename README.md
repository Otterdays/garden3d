# Garden3D

A cozy 3D gardening game built with Vite, Three.js (WebGPU), and TypeScript.

## Features
- **Deterministic Grid**: Farming system with isometric grid alignment.
- **Dynamic Environment**: Time of day, weather, and lush vegetation.
- **NPC System**: Autonomous entities with wandering AI.
- **Economic Loop**: Shop for seeds and sell harvest.
- **Advanced UX**: Interactive hotbar, tool switching, and smooth camera controls.

## Tech Stack
- **Engine**: Three.js (WebGPU Renderer)
- **Shader**: TSL (Three.js Shading Language) for wind effects.
- **Build Tool**: Vite
- **Language**: TypeScript

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
